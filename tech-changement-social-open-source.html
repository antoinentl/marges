Les outils de changement social open source

Quelques exemples d’outils web, principalement open source (et donc, réutilisable, adaptable et améliorable), pour booster d’une façon ou d’une autre le changement social.

Ressources générales

  • Open Government Partnership Toolbox
  • TOOL SELECTION ASSISTANT

Civic Tech

Informer

  • DKAN, par Nücivic (Etats-Unis) : portail d’open data à destination des collectivités.

  • 2 minutes 4 my city, par Open Knowledge Estonia (Estonie) : application qui connecte les citoyens avec les actualités et décisions locales

Mobiliser

  • Meu Rio (Brésil): plateforme de lobby citoyen
  • We the people (Etats-Unis) : application officielle de pétitions de l’administration Obama

Contribuer

  • Fix my Street, par mySociety.org (Royaume-Uni) : plateforme open source de signalement.
  • Discourse

Décider

  • Democracy OS (Argentine) : plateforme de consultation
  • Consul (Espagne) : portail municipale pour les concertations et le budget participatif (cf par ex decide.madrid.es)
  • Loomio

AMAP et groupements d’achats

  • www.cagette.net : Une application libre pour développer le circuit court / le logiciel libre des AMAP et groupements d’achat citoyens. Projet Open Source, vend l’hébergement

Portails et annuaires à creuser

  • Tool box de D-cent
  • code.gov : portail qui libère le code de toutes les plateformes gouvernementales américaines.
  • ogptoolbox
